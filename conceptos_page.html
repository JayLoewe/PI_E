Conceptos Fundamentales

Wireframes de Baja Fidelidad

Representaciones básicas que se enfocan en la estructura y layout general, sin detalles visuales. Ideales para las primeras etapas de diseño.

Principiante Estructura 🔗Quiero leer mas

Wireframes de Alta Fidelidad

Diseños detallados que incluyen elementos específicos, contenido real y una representación más precisa del producto final.

Intermedio Detalle 🔗Ver prototipos de alta y baja calidad

Arquitectura de Información

La organización, estructuración y etiquetado del contenido de manera efectiva y sostenible para ayudar a los usuarios a encontrar información.

Intermedio Organización 🔗Como empezar

User Flow

El camino que sigue un usuario para completar una tarea en tu sitio web o aplicación. Esencial para wireframes funcionales.

Principiante Navegación 🔗Guía definitiva

Grid Systems

Sistemas de rejillas que proporcionan estructura y consistencia al layout, facilitando la organización del contenido.

Avanzado Layout 🔗Conoce más

Responsive Design

Enfoque de diseño que hace que las páginas web se vean bien en todos los dispositivos y tamaños de pantalla.
